nathantranquilla.me exists in part as an outlet for trying cool new web stuff. One of the things I've been interested in trying out is Web Components in the context of a shared elements library or for the purpose of encapsulating design decisions. In my case, I decided to use lean design principles and encapsulate them in a shared blog components library which I'm using in this site. Check out the source code for this page and look for tags prefixed with lbwc to see them. eg. < lbwc-app-bar />

It also gave me the chance to try out a new static site generator, eleventy. In the past I had used Gatsby, but it is much more opinionated and assumes you'll do the standard one-page app with React. In this case, simpler is better, so I decided to go with a multi-page application with few or optimized resources for quick page loads. If you want to check out the code, you can view the github repo here.
